Short description:
Long description: The official GRI-Mech 3.0 has the minimum temperature on the NASA polynomial at 300K. This prevents it from being used to evaluate the standard properties at 298K as required by some parts of RMG. Extrapolating 2 degrees beyond the the recommended range probably introduces less error than not using the thermo at all, so the range has been extended to 298K.
